# web-parsing
Этот репозиторий содержит проекты парсинга сайтов используя Beautiful Soup.


Первый проект:
Сайт http://list.infox.by/ получить список названий фирм. Для каждой фирмы получить УНН, адрес, телефон,
сайт, e-mail. Провести очистку данных: дублированные фирмы не должны присутствовать в списке.
